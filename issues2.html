Git remote

Looge kaug-reposiit

  • Minge GitHubi (või GitLabi, Bitbucketi) ja looge uus reposiit

Algatage kohalik Git reposiit: Oma projekti kaustas käivitage järgmised käsud

git init git add . git commit -m "Algne commit"

Lisage kaug-reposiit: Kopeerige oma kaug-reposiidi URL ja käivitage järgmine käsk

git remote add origin https://github.com/username/repo_name.git

Saadke muudatused kaug-reposiiti

git push -u origin master

Nüüd on teie kohalik reposiit ühendatud kaug-reposiidiga ja muudatusi saab edaspidi saata käsuga git push
